The RSI Loop · Architecture

Two-stage validation for self-improving AI

Accuracy can be optimised. Compliance cannot be self-certified.

Stage 01 — Accuracy

Does the detector classify posture correctly?

Inputs
detector.py + benchmarks.json (10 labelled scenarios)
Output
accuracy · precision · recall · F1
Gate accuracy ≥ 90% → if PASS, hand off to Stage 2
▼

Stage 02 — Compliance

Are the AI-tuned thresholds clinically plausible?

Inputs
Live thresholds vs. Clinical Gold Standard (forward-head 15–25°, wrist 40–60°)
Output
PASS · WARNING · HARD FAILURE per threshold
Gate every threshold = PASS → if PASS, accept the iteration
